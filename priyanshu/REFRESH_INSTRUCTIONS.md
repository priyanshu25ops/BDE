# How to Reload the Site

After any changes, reload the site using one of these methods:

## Method 1: Browser Refresh (Easiest)
- Press `Ctrl + R` (Windows) or `Cmd + R` (Mac)
- Or press `F5`
- Or click the refresh button in your browser

## Method 2: Hard Refresh (Clears Cache)
- Press `Ctrl + Shift + R` (Windows) or `Cmd + Shift + R` (Mac)
- This ensures you get the latest changes

## Method 3: Auto-Reload
If you want automatic reloading, you can:
1. Open browser DevTools (F12)
2. Right-click the refresh button
3. Select "Empty Cache and Hard Reload"

## Quick Refresh Shortcut
Just press `F5` anytime you make changes!
